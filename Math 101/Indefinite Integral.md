\int f(x) \, dx 
$$
Table of Antiderivatives

| Function $f(x)$              |  Indefinite integral $\int f(x) \, dx$  |
| ---------------------------- |:---------------------------------------:|
| $1$                            |                   x+C                   |
| $x^n (n\neq-1)$              |        $\frac{x^(n+1)}{n+1} +C$         |
| $\frac{1}{x}$                | $\ln \begin{vmatrix}x\end{vmatrix} + C$ |
| $e^x$                        |                $e^x + C$                |
| $\sin x$                     |              $-\cos(x)+C$               |
| $\cos x$                     |              $\sin(x) + C$              |
| $\sec^2(x)$                  |               $\tan x +C$               |
| $\frac{1}{(\sqrt{ 1-x^2 })}$ |               $\arcsin+C$               |
| $\frac{1}{(1+x^2)}$          |             $\arctan(x)+ C$             |
